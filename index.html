Dummy is a Middleman project to try and test things quickly.

Installation

$ git clone git@github.com:bitaculous/dummy.git dummy
$ cd dummy
$ bundle

Usage

  1. Boot up Middleman

    $ middleman

  2. Open your browser at http://0.0.0.0:4567 and start hacking
